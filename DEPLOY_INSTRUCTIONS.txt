=== Deployment to Mimoria Server ===

QUICK DEPLOY (One Command):
---------------------------
./quick_deploy.sh

This will:
1. Clone/update the repo on server
2. Install dependencies
3. Setup systemd service
4. You just need to edit .env and start service

FULL DEPLOY (With HTTPS):
-------------------------
./deploy_to_mimoria.sh

This will:
1. Copy deployment scripts
2. Deploy application
3. Setup HTTPS (if domain provided)

MANUAL STEPS AFTER DEPLOYMENT:
------------------------------
1. SSH to server: ssh mimoria@100.126.32.26
2. Edit .env: nano ~/llm-analysis-quiz/.env
   - Add your OPENROUTER_API_KEY
3. Start service: sudo systemctl start llm-quiz
4. Check status: sudo systemctl status llm-quiz
5. View logs: sudo journalctl -u llm-quiz -f

FOR HTTPS:
----------
1. Point your domain to: 65.109.236.39
2. On server: cd ~/llm-analysis-quiz && ./setup_https.sh
3. Enter your domain when prompted

YOUR ENDPOINT:
-------------
HTTP:  http://65.109.236.39:8000/quiz
HTTPS: https://your-domain.com/quiz (after HTTPS setup)
